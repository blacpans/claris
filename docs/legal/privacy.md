# プライバシーポリシー (Privacy Policy)

Claris開発チーム（以下「当チーム」といいます。）は、本アプリケーション「Claris」（以下「本サービス」といいます。）において、利用者の個人情報の取り扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」といいます。）を定めます。

## 1. 収集する情報
本サービスでは、以下の情報を取得・利用します。
- **Google アカウント情報**: メールアドレス、氏名、プロフィール画像（認証および本人確認のため）
- **アクセス許可（Google OAuth スコープ）**: Gmail, Google Calendar, Google Maps, Google Drive 等、利用者が本サービス内で許可した機能の実行に必要なデータ
- **対話内容**: Gemini API との対話、およびセッション維持のためのログ情報

## 2. 利用目的
取得した情報は、以下の目的で利用されます。
- 自律型エージェントによるアクションの実行（メール確認、予定登録、コードレビュー等）
- 利用者のコンテキストに基づいたパーソナライズされた提案の作成
- 本サービスの改善、不具合の修正

## 3. 第三者への提供
本サービスは、以下のパートナーサービスを除き、本人の同意なく第三者に個人情報を提供することはありません。
- **Google Cloud / Vertex AI**: エージェントの思考（Gemini API）およびデータ保存（Firestore等）のため
- **GitHub**: ソースコードの連携およびレビュー機能のため

## 4. データの保存と削除
- 利用者のセッションデータおよび認証情報は、Google Cloud Firestore の安全なデータベースに保存されます。
- 利用者は、本サービス内のログアウトまたは Google アカウント設定の「サードパーティ製のアプリとサービス」から、いつでもアクセス許可を管理・取り消すことができます。

## 5. お問い合わせ
本ポリシーに関するお問い合わせは、開発者（blacpans@gmail.com）までお願いいたします。
